# Task Manager App

Aplicativo de gerenciamento de tarefas para organização pessoal e produtividade, com foco em usabilidade e responsividade.

## Tecnologias Utilizadas
- C#
- .NET Core
- SQL Server
- Razor Pages

## Funcionalidades
- Cadastro e edição de tarefas
- Marcar tarefas como concluídas
- Filtragem por status (pendente/concluído)
- Interface adaptada para dispositivos móveis
